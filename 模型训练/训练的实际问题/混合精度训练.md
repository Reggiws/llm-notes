# 混合精度训练

![[Pasted image 20260603110143.png]]

| 概念 | 含义 | 精度 |
| :--- | :--- | :--- |
| **Master Weight** | 优化器维护的高精度权重 | FP32 |
| **Working Weight** | 前向/反向实际用的权重 | FP16 |
| **Shadow Weight**（不同语境） | 某些优化器（如动量）维护的历史值 | 同精度或更高 |
### Loss Scaling
![[Pasted image 20260603110756.png]]
## FP16 如何节省显存


- **激活值用 FP16**：大小直接减半。
    
- **前向传播用 FP16 权重**：从 Master FP32 临时转换而来，反向传播后就丢弃，不永久占用显存。
    

**结果**：虽然你保留了 FP32 的 Master Weight 和优化器状态，但**最大的显存消耗者——激活值——节省了 50%**。
